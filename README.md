# Nuclei - a Course Selection Web App

A web app that streamlines course selection and appeals for students, attempting to create a better course selection website than Talos, the one that Bronx Science was using.

## Tech Stack
- **Backend:** Node.js, Express
- **Frontend:** Materialize, Bootstrap
- **Database:** Google Firebase

## Features
- Students can select their courses, appeal for classes, and view the course guide.
- Administrators and counselors can view and manage student selections and appeals.
